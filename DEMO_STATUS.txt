PHYSICS SIMULATION PIPELINE
  1. Neuron Generation
      Random 3D parametric curves (cubic Bezier-like)
      Action potential as moving Gaussian pulse
      Multiple neurons in configurable volume
  
  2. Magnetic Field Computation
      Discretized Biot-Savart law (vectorized numpy)
      2D sensor grid on NV layer (z=0 plane)
      Time-series B-field at each sensor
  
  3. ODMR Sensing
      Frequency-shift proxy from B-field projection
      Realistic noise models (shot, thermal, drift)
      Clean + noisy time-series output
  
  4. Graph Construction
      Node features: frequency shift, temporal derivative, spatial mean
      Edges: spatial proximity + temporal continuity
      Weighted by distance and edge correlation
  
  5. Denoising
      Gaussian filtering (spatial + temporal)
      Preserves signal morphology

================================================================================
                               TESTED ENDPOINTS
================================================================================

 POST /odmr
  Request: {"n_neurons": 50, "sensor_res": 16, "n_time": 20, "noise_level": 0.2}
  Response: df_clean, df_noisy (time-series), grid coords, times
  
 POST /graph
  Request: {"n_neurons": 40, "sensor_res": 16, "n_time": 20}
  Response: nodes (180), edges (144), node features, edge weights
  
 POST /denoise
  Request: {"n_neurons": 40, "sensor_res": 16, "n_time": 20}
  Response: df_denoised (Gaussian filtered), before/after comparison

                             PERFORMANCE NOTES

 Biot-Savart: vectorized over all segments & sensors, ~16x16 sensor grid

Event Flow:
  User clicks "Run Simulation" 
     ControlPanel POSTs to http://localhost:8000/odmr
     ControlPanel dispatches "simulation:clean" & "simulation:noisy" events
     GNNVisualizer3D listens to events & extracts playback scalars
     Animation loop modulates node intensity from playback scalars
     Smooth playback of backend simulation
